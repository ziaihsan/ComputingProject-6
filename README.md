# Crypto Dual Heatmap (EMA + RSI Multi-Layer)

## Quick Start

```bash
# 1. Install (sekali saja)
pip3 install uvicorn fastapi aiohttp pandas numpy

# 2. Run
python3 run.py
```

## Fitur
- RSI Heatmap multi-timeframe (15m, 1h, 4h, 12h, 24h)
- Sinyal Long & Short dengan 5 layer kekuatan
- Data real-time dari Binance API

## Tech Stack
Backend: Python, FastAPI | Frontend: HTML, TailwindCSS | Data: Binance API
